# aiml-crash-uzefa

Python practice tasks from Day 2 of AI/ML training.

## Files
- intro.py — Personal introduction using variables and f-strings
- skills_counter.py — Prints numbered list of skills
- even_odd.py — Checks if a number is even, odd, or zero
- tip_calculator.py — Calculates tip and total bill
- word_frequency.py — Counts word frequency in a sentence
- calculator.py — Basic calculator with 4 operations
- grade_classifier.py — Assigns letter grades to students
- guessing_game.py — Number guessing game (1-100)
- contact_book.py — Mini contact book with search

## How to run
python filename.py
